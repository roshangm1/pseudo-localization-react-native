# pseudo-localization-rn

Pseudo Localization for react native

## Installation

```sh
npm install pseudo-localization-rn
```

## Usage

```js
import PseudoLocalizationRn from "pseudo-localization-rn";

// ...

const result = await PseudoLocalizationRn.multiply(3, 7);
```
